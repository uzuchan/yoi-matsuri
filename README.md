# 宵祭(よいまつり) / yoi-matsuri

日本の夏祭りの夜を体験できる、インタラクティブな Web ゲーム。
夜の神社の参道を歩き、金魚すくいの屋台で遊べます。提灯・花火・祭囃子・群衆のざわめき・虫の声で「夏祭りの夜」に包まれる体験を目指しています。

すべてブラウザ上で動作し、**外部アセットを一切使わずプロシージャル生成**(3D は Three.js、音は Web Audio API で合成)しています。

## あそびかた

1. 夜の参道を歩く(`WASD` / 矢印キー、またはマウス長押しで前進)
2. 金魚すくいの屋台に近づく(`E` またはクリックで話しかける)
3. 店主と会話して「遊んでいく」を選ぶ
4. 金魚すくい — ポイをそっと動かして金魚をすくう。**速く動かすと水の抵抗で紙が破れる**ので慎重に
5. 結果に応じた店主の反応と報酬を受け取り、参道へ戻る

マウスのみ・キーボードのみ、どちらでも最後まで遊べます。

## 開発・実行

```bash
cd yoi-matsuri
npm install
npm run dev        # 開発サーバー(ブラウザで開く)
npm run build      # 本番ビルド
npm run preview    # ビルド結果をプレビュー
```

### 品質チェック

```bash
npm run typecheck  # 型チェック(TypeScript strict)
npm run lint       # ESLint
npm run test       # ユニットテスト(Vitest)
npm run test:e2e   # E2Eテスト(Playwright)
```

## 技術スタック

- **Vite + React 19 + TypeScript(strict)**
- **Three.js** — 夜の参道・金魚すくいの 3D 描画(react-three-fiber は不使用。固定タイムステップの自前ゲームループで制御)
- **Web Audio API** — 祭囃子・虫の声・群衆・効果音をすべてプロシージャル合成(外部音声ファイル不使用)
- **Vitest / Playwright** — ユニット / E2E テスト

状態管理は自前の軽量ステートマシン + 型付き EventBus。外部の状態管理ライブラリや音声ライブラリは使っていません。

## プロジェクト構成

```
yoi-matsuri/          # アプリ本体(Vite プロジェクト)
├── src/
│   ├── core/         # ゲームループ・シーン管理・入力・イベントバス
│   ├── game/         # 純TSのドメインロジック(金魚すくいの物理・会話・結果)
│   ├── scenes/       # 参道 / 会話 / 金魚すくい / 結果 の各シーン
│   ├── world/        # 提灯・鳥居・屋台・群衆・花火などの環境
│   ├── audio/        # Web Audio によるプロシージャル音響
│   └── ui/           # React の HUD・会話・結果オーバーレイ
├── tests/            # ユニットテスト
└── e2e/              # E2Eテスト
docs/                 # 設計ドキュメント(ゲームデザイン・アート・技術・他)
```

## プライバシー

このゲームは**ユーザーのデータを一切収集・保存・送信しません**(ネットワーク通信・localStorage・Cookie を使用していません)。すべてブラウザ内で完結します。
